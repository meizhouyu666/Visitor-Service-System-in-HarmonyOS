# 发布与协作门禁（零配置安全）

## 发布门禁（必须全部通过）
1. 组员拉取最新代码后，沿用现有步骤可直接启动（无需改本地配置）。
2. 既有核心流程保持可用：
   - 登录流程
   - 投诉主链路
   - 应急主链路
   - 既有查询接口（`/api/query/*`）
3. 新增能力完成验证并在文档中说明。
4. 本次提交必须更新 README 版本说明。

## PR 检查清单（可直接复制到 PR 描述）
- [ ] 不需要组员侧配置变更
- [ ] 既有启动命令不变
- [ ] 既有 API 路由保持可用
- [ ] Flyway 迁移仅做增量
- [ ] 已包含兼容性说明
- [ ] 已包含回滚方案
- [ ] 已更新 README 版本说明
