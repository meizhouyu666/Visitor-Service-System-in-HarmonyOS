# HarmonyOS Frontend Skeleton

This folder contains the HarmonyOS (ArkTS + ArkUI + Stage Model) skeleton for the Visitor Service System.

## Modules Covered

- Complaint system
- Emergency information
- Star hotel / non-star hotel
- Scenic spots and routes
- Dining / performances
- Weather / traffic

## Notes

- API base URL is currently set in `ApiClient.ets`.
- The project is structured for Stage Model (`UIAbility`) and typed ArkTS usage.
- Pages are implemented as modular skeleton pages ready for API binding.
